✨ Sala 1: Enviar Sinais com Tags Básicas

Tags HTML são como sinais de rádio que definem a estrutura da sua interface. Veja:

<p>Olá, astronauta!</p>

Como usar: Tags começam com <nome> e terminam com </nome>. O <p> cria parágrafos.

Desafio 1: O que aparece ao usar <p>Missão HTML</p>?

Desafio 2: Qual é a tag para um parágrafo?

🎉 Ótimo trabalho! Você enviou seus primeiros sinais. Vamos para a próxima sala! 🚪

🚀 Sala 2: Estruturar a Nave com a Tag html

A tag <html> é como o casco da nave, envolvendo todo o conteúdo. Veja:

<html>
  <p>Painel de Controle</p>
</html>

Como usar: Todo documento HTML começa com <!DOCTYPE html> e usa <html> como base.

Desafio 3: Qual é a primeira linha de um documento HTML?

Desafio 4: O que aparece? 

<html>
  <p>Estação Espacial</p>
</html>

🎉 Perfeito! A estrutura da nave está pronta. Continue, astronauta! 🚀

🖥️ Sala 3: Configurar o Painel com head e meta

A tag <head> configura o painel de controle com metadados. Veja:

<head>
  <meta charset="UTF-8">
  <title>Painel Estelar</title>
</head>

Como usar: O <meta charset="UTF-8"> define a codificação, e <title> define o título da aba.

Desafio 5: O que faz <meta charset="UTF-8">?

Desafio 6: O que aparece na aba do navegador? 

<head>
  <title>Nave Save</title>
</head>

🎉 Boa, astronauta! O painel está configurado. Vamos adiante! 🖥️

🛠️ Sala 4: Organizar o Cockpit com body

A tag <body> é o cockpit, onde o conteúdo visível da nave aparece. Veja:

<body>
  <p>Bem-vindo à nave!</p>
</body>

Como usar: Tudo visível na página vai dentro de <body>.

Desafio 7: O que aparece? 

<body>
  <p>Sistema Online</p>
</body>

Desafio 8: Qual tag contém o conteúdo visível?

🎉 Você é demais! O cockpit está pronto. Próxima sala! 🛠️

📜 Sala 5: Exibir Mensagens com p e h1-h6

As tags <p> e <h1> a <h6> mostram textos, como comunicadores estelares. Veja:

<h1>Missão Estelar</h1>
<p>Iniciar contagem!</p>

Como usar: <h1> é o título maior, <h6> o menor. <p> é para parágrafos.

Desafio 9: O que aparece? 

<h2>Comando Ativado</h2>

Desafio 10: Qual tag cria o maior título?

Desafio 11: Quantas tags de título existem?

🎉 Comunicação clara! Você domina textos. Vamos continuar! 📜

⚙️ Sala 6: Adicionar Detalhes com Atributos

Atributos são como configurações de equipamentos, dando mais funções às tags. Veja:

<p id="comunicador">Mensagem</p>

Como usar: Atributos como id ou class vão dentro da tag inicial, como <tag id="valor">.

Desafio 12: Qual é o atributo em <p class="estelar">Texto</p>?

Desafio 13: O que aparece? 

<p id="alerta">Emergência!</p>

🎉 Equipamento configurado! Você domina atributos. Próxima sala! ⚙️

🔗 Sala 7: Navegar com Links a

A tag <a> cria links, como portais espaciais para outras páginas. Veja:

<a href="https://nasa.gov">Visitar NASA</a>

Como usar: O atributo href define o destino do link.

Desafio 14: O que aparece? 

<a href="galaxia.html">Explorar</a>

Desafio 15: Qual atributo define o destino do link?

🎉 Portal ativado! Você domina links. Vamos adiante! 🔗

🖼️ Sala 8: Exibir Imagens Estelares com img

A tag <img> mostra imagens, como visores estelares. Veja:

<img src="nave.jpg" alt="Nave Koda">

Como usar: Use src para o caminho da imagem e alt para descrição.

Desafio 16: Qual atributo define a descrição da imagem?

Desafio 17: O que acontece se a imagem não carrega? 

<img src="erro.jpg" alt="Planeta">

🎉 Visor estelar! Você domina imagens. Próxima sala! 🖼️

📋 Sala 9: Listar Recursos com ul e ol

As tags <ul> e <ol> criam listas, como inventários de recursos. Veja:

<ul>
  <li>Propulsor</li>
  <li>Radar</li>
</ul>

Como usar: <ul> faz listas não ordenadas (pontos), <ol> faz listas numeradas.

Desafio 18: O que aparece? 

<ol>
  <li>Missão 1</li>
  <li>Missão 2</li>
</ol>

Desafio 19: Qual tag cria uma lista com marcadores?

🎉 Inventário pronto! Você domina listas. Vamos adiante! 📋

🗂️ Sala 10: Organizar com div e span

As tags <div> e <span> organizam o conteúdo, como módulos da nave. Veja:

<div>
  <span>Alerta</span>: Sistema ativado
</div>

Como usar: <div> agrupa blocos, <span> marca partes de texto.

Desafio 20: O que aparece? 

<div>
  <span>Status:</span> Online
</div>

Desafio 21: Qual tag agrupa blocos de conteúdo?

🎉 Módulos organizados! Você domina div e span. Próxima sala! 🗂️

🏛️ Sala 11: Estruturas Semânticas

Tags semânticas como <header>, <footer> dão significado, como setores da nave. Veja:

<header>
  <h1>Painel Principal</h1>
</header>

Como usar: Use <header>, <footer>, <article>, <section> para clareza.

Desafio 22: Qual tag define o rodapé da página?

Desafio 23: O que aparece? 

<section>
  <p>Relatório Estelar</p>
</section>

🎉 Setores definidos! Você domina semântica. Próxima sala! 🏛️

📬 Sala 12: Coletar Dados com Formulários

A tag <form> coleta dados, como um terminal de comunicação. Veja:

<form>
  <input type="text" name="nome">
</form>

Como usar: Use <input> com type para texto, número, etc.

Desafio 24: Qual tag cria um campo de texto?

  <input type="text" name="astronauta">
</form>

Desafio 25: Qual atributo define o tipo de entrada?

🎉 Terminal ativo! Você domina formulários. Próxima sala! 📬

🛡️ Sala 13: Proteger o Código com Validação

Atributos como required protegem formulários, como escudos de energia. Veja:

<form>
  <input type="text" required>
</form>

Como usar: required impede envios sem preenchimento.

Desafio 26: O que faz required em um <input>?

Desafio 27: O que acontece se o campo está vazio? 

<input type="text" required>

🎉 Escudo ativado! Seu código está protegido. Vamos para a final! 🛡️

🌌 Sala 14: Interface Web Galáctica

Combine tudo para criar a Interface Web Galáctica! Veja:

<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Nave Save</title>
</head>
<body>
  <header>
    <h1>Missão HTML</h1>
  </header>
  <section>
    <form>
      <input type="text" name="nome" required>
    </form>
  </section>
</body>
</html>

Como usar: Combine tags semânticas, formulários e validação para uma interface completa.

</body>

Desafio 31: Qual atributo impede envio de formulário vazio?
