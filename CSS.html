✨ Sala 1: Ativar CSS no Painel

CSS estiliza o HTML, como controles visuais da nave. Conecte CSS ao HTML com a tag <link>.

<head>
  <link rel="stylesheet" href="style.css">
</head>

Como usar: Use <link rel="stylesheet" href="arquivo.css"> no <head>.

🔍 Sala 2: Selecionar Elementos Estelares

Seletores CSS são como radares que encontram elementos HTML. Exemplo:

p {
  color: blue;
}

Como usar: Seletores como p, .class, #id aplicam estilos.

🎨 Sala 3: Controlar Cores Estelares

Propriedades como color e background-color pintam a interface. Veja:

.painel {
  background-color: black;
  color: white;
}

Como usar: Use nomes de cores, hex, RGB, etc.

Desafio 5: O que faz color: blue;?

Desafio 6: Qual propriedade muda o fundo?

✍️ Sala 4: Estilizar Textos Cósmicos

Propriedades como font-size, font-family controlam textos. Veja:

h1 {
  font-size: 24px;
  font-family: Arial;
}

Como usar: Ajuste tamanho, fonte e peso do texto.

Desafio 7: O que faz font-size: 16px;?

📏 Sala 5: Gerenciar o Box Model

O Box Model controla margens, bordas e preenchimento. Veja:

.modulo {
  margin: 10px;
  padding: 20px;
  border: 1px solid white;
}

Como usar: margin é o espaço externo, padding interno, border é a borda.

Desafio 9: O que faz padding: 15px;?

Desafio 10: Qual propriedade define a borda?

Desafio 11: Qual propriedade controla o espaço externo?

🎉 Espaço controlado! Você domina o Box Model. Vamos continuar! 📏

📍 Sala 6: Posicionar Módulos

A propriedade position move elementos, como módulos da nave. Veja:

.nave {
  position: absolute;
  top: 10px;
}

Como usar: Use position: static, relative, absolute, fixed com top, left, etc.

Desafio 12: O que faz position: fixed;?

Desafio 13: Qual propriedade define a posição vertical?

🎉 Módulos posicionados! Você domina posição. Próxima sala! 📍

🤝 Sala 7: Alinhar com Flexbox

Flexbox alinha elementos, como painéis da nave. Veja:

.container {
  display: flex;
  justify-content: center;
}

Como usar: display: flex; ativa o flexbox, justify-content alinha horizontalmente.

Desafio 14: O que faz justify-content: space-between;?

Desafio 15: Qual propriedade ativa o flexbox?

🎉 Alinhamento perfeito! Você domina flexbox. Vamos adiante! 🤝

📐 Sala 8: Organizar com Grid

CSS Grid organiza layouts, como setores da nave. Veja:

.grid {
  display: grid;
  grid-template-columns: 1fr 1fr;
}

Como usar: display: grid; ativa o grid, grid-template-columns define colunas.

Desafio 16: O que faz grid-template-columns: 1fr 1fr;?

Desafio 17: Qual propriedade ativa o grid?

🎉 Layout perfeito! Você domina grid. Próxima sala! 📐

Propriedades como box-shadow e opacity criam efeitos. Veja:

  opacity: 0.8;

Como usar: box-shadow adiciona sombras, opacity controla transparência.

Desafio 18: O que faz box-shadow: 0 0 5px blue;?

Desafio 19: O que faz opacity: 0.5;?

🎉 Efeitos estelares! Você domina visuais. Vamos adiante! ✨

Pseudo-classes como :hover estilizam links. Veja:

Como usar: :hover aplica estilos ao passar o mouse.

📱 Sala 12: Responsividade Cósmica

Media queries adaptam a interface para diferentes telas. Veja:

  .painel {
    font-size: 14px;
  }
}

Como usar: Use @media para ajustar estilos por tamanho de tela.

Desafio 24: O que faz @media (max-width: 800px)?

🎉 Interface adaptada! Você domina responsividade. Próxima sala! 📱

🌠 Sala 13: Animações Estelares

Animações com @keyframes e animation dão vida à interface. Veja:

@keyframes pulsar {
  0% { opacity: 1; }
  100% { opacity: 0; }
}
.pulsar {
  animation: pulsar 2s infinite;
}

Como usar: Defina @keyframes e aplique com animation.

Desafio 26: O que faz animation: girar 1s infinite;?

Desafio 27: Como definir uma animação?
