🛡️ Sala 13: Proteger o Código com try/except

Use try e except para proteger seu código de erros, como um escudo ninja. Veja:

try:
    numero = int(input("Digite um número: "))
    print(numero * 2)
except ValueError:
    print("Apenas números!")

Como usar: O try tenta o código; se der erro, o except mostra uma mensagem.

Desafio 27: Se o usuário digitar abc, o que aparece?

🌌 Sala 14: Calculadora Estelar

Combine tudo para criar a Calculadora Estelar! Veja:

def calcular(op, n1, n2):
    if op == "+":
        return n1 + n2
    elif op == "-":
        return n1 - n2
    return "Operação errada"
try:
    op = input("Operação (+ ou -): ")
    n1 = int(input("Número 1: "))
    n2 = int(input("Número 2: "))
    print("Resultado:", calcular(op, n1, n2))
except ValueError:
    print("Use números!")

Como usar: A função calcular usa if para somar ou subtrair. O try protege contra erros.
